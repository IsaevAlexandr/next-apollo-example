# SSR (Next.js) + Apollo

[Open app](https://rick-and-morty-party.sany188.now.sh)

Simple `next.js` + `apollo` app example with async data fetching on client (`@apollo/react-hooks`) and some business logic.

## Quick start

```sh
$ npm i
$ npm run build
$ npm start
```

## Development

```sh
$ npm i
$ npm run dev
```
